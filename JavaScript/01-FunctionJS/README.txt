Definicion de funciones: 
    En pocas palabras una funcion es un vinculo entre un valor y su resultado.
    Este valor puede ser nulo, por lo que una funcion puede recibir todos o ningun valor
    pero siempre devolvera un resultado.

Como invocar un funcion en JavaScript:

    const cuadrado = function( x ) {
        return x * x;
    };
